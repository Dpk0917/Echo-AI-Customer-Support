# Echo AI Customer Support

Echo AI is a Next.js-based customer interaction system that provides intelligent, real-time chatbot responses.

## Features

- Real-time chat interface
- AI-powered responses
- Easy embed support for external websites
- Dashboard and test pages for development

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Project Structure

- `src/app` - Next.js app routes and API handlers
- `src/Components` - UI components
- `public` - Static assets and browser scripts
- `demoweb-html` - Demo HTML embedding assets
